## About the extension

Extension to speed up Zillow listing review with a floating, sleek quick-scan panel.

When opening a listing page, you now get a floating widget that:

- shows live feature results in a `Results` tab,
- lets you choose what matters in a `Settings` tab,
- can be collapsed into a small edge tab and reopened anytime.

### Default feature checks

- In-Unit Laundry
- Central Air
- Dishwasher
- Dedicated Parking
- Pet Friendly

Selections persist automatically per browser profile.

## Install in Chrome (Unpacked)

1. Download or clone this folder locally.
2. Open Chrome and go to `chrome://extensions`.
3. Turn on **Developer mode** (top-right).
4. Click **Load unpacked**.
5. Select the `zillow-extension` folder.
6. Open a Zillow listing page and the floating panel will appear automatically.

### Updating after code changes

- After editing files, return to `chrome://extensions`.
- Find **Zillow Feature Checker** and click the refresh/reload icon.
